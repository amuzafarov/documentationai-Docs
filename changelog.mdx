## Upgrading Compitio SDK

<Callout kind="info">
Review the [migration guide](https://docs.example.com/migration) for detailed steps. Test in staging before production deployment.
</Callout>

Follow these steps to upgrade your Compitio integration.

<Steps>
  <Step title="Backup Current Setup" icon="database">
    Export your current configuration and data.

    ```bash
    compitio-cli export --file=backup.json
    ```
  </Step>
  <Step title="Update SDK" icon="download">
    Install the latest version using your package manager.

    <CodeGroup tabs="npm,yarn">
    ````bash
    npm install compitio-sdk@latest
    ````

    ````bash
    yarn add compitio-sdk@latest
    ````
    </CodeGroup>
  </Step>
  <Step title="Update Configuration" icon="settings">
    Replace old endpoints with new ones.

    ````javascript
    // Old
    const client = new CompitioClient({ baseURL: 'https://api.example.com/v0' });

    // New
    const client = new CompitioClient({ baseURL: 'https://api.example.com/v1' });
    ````
  </Step>
  <Step title="Test Integration" icon="check-circle">
    Run your test suite to verify functionality.
  </Step>
</Steps>
